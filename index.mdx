franz-go is an all-encompassing Apache Kafka client written entirely in Go. It supports every Kafka feature from version 0.8.0 through 4.1+, including exactly-once semantics, cooperative consumer group rebalancing, all SASL mechanisms, all compression codecs, and a high-level admin API. If a client KIP exists, franz-go supports it.

## Why franz-go?

franz-go is the fastest and most memory-efficient Kafka client in Go — up to 4x faster at producing and 10–20x faster at consuming compared to alternatives. It is written entirely in pure Go with no C bindings, making it easy to cross-compile and deploy anywhere.

## Compatible brokers

franz-go works with any Kafka-compatible broker:

- **Apache Kafka** — all versions from 0.8.0 through 4.1+
- **Redpanda** — the fastest Kafka-compatible streaming platform
- **Confluent Platform** — enterprise Kafka distribution
- **Amazon MSK** — managed Kafka on AWS
- **Microsoft Event Hubs** — Azure's Kafka-compatible service

## Get started

<Steps>
  <Step title="Install the main package">
    ```bash
    go get github.com/twmb/franz-go
    ```
  </Step>
  <Step title="Create a client and produce a record">
    ```go
    cl, err := kgo.NewClient(
        kgo.SeedBrokers("localhost:9092"),
    )
    if err != nil {
        panic(err)
    }
    defer cl.Close()

    record := &kgo.Record{Topic: "my-topic", Value: []byte("hello")}
    if err := cl.ProduceSync(ctx, record).FirstErr(); err != nil {
        panic(err)
    }
    ```
  </Step>
  <Step title="Poll for records">
    ```go
    for {
        fetches := cl.PollFetches(ctx)
        fetches.EachRecord(func(r *kgo.Record) {
            fmt.Printf("offset %d: %s\n", r.Offset, r.Value)
        })
    }
    ```
  </Step>
</Steps>

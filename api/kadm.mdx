The `kadm` package provides an opinionated, high-level Kafka admin client built on top of a `*kgo.Client`. It covers the most common administrative tasks — creating and listing topics, managing consumer group offsets, altering configs, and working with ACLs — in a convenient API that hides the complexity of the underlying Kafka protocol. For operations that `kadm` does not cover, drop down to the `kmsg` package and issue raw protocol requests directly.

```bash
go get github.com/twmb/franz-go/pkg/kadm
```

```go
import "github.com/twmb/franz-go/pkg/kadm"
```

## Client

`kadm.Client` is a thin wrapper around a `*kgo.Client`. You can create one from an existing kgo client or directly from kgo options.

```go
// From an existing kgo.Client:
kgocl, _ := kgo.NewClient(kgo.SeedBrokers("localhost:9092"))
admcl := kadm.NewClient(kgocl)
defer admcl.Close()

// Directly from options:
admcl, err := kadm.NewOptClient(kgo.SeedBrokers("localhost:9092"))
```

| Function | Signature | Description |
|---|---|---|
| `NewClient` | `func NewClient(cl *kgo.Client) *Client` | Wraps an existing kgo client. |
| `NewOptClient` | `func NewOptClient(opts ...kgo.Opt) (*Client, error)` | Creates a kgo client and wraps it. |
| `Close` | `func (cl *Client) Close()` | Closes the underlying kgo client. |
| `SetTimeoutMillis` | `func (cl *Client) SetTimeoutMillis(millis int32)` | Overrides the default 15 s request timeout. |

## ACLs

`kadm` uses an `ACLBuilder` to construct ACL operations. Build the desired principal, host, resource, and operations, then call `CreateACLs`, `DescribeACLs`, or `DeleteACLs`.

```go
b := kadm.NewACLs().
    Allow("User:alice").
    AllowHosts("*").
    Topics("my-topic").
    Operations(kadm.OpRead, kadm.OpDescribe)

err := admcl.CreateACLs(ctx, b)
```

| Function | Signature | Description |
|---|---|---|
| `CreateACLs` | `func (cl *Client) CreateACLs(ctx context.Context, b *ACLBuilder) (CreateACLsResults, error)` | Creates ACLs matching the builder. |
| `DescribeACLs` | `func (cl *Client) DescribeACLs(ctx context.Context, b *ACLBuilder) (DescribeACLsResults, *ShardErrors, error)` | Returns ACLs that match the builder's filter. |
| `DeleteACLs` | `func (cl *Client) DeleteACLs(ctx context.Context, b *ACLBuilder) (DeleteACLsResults, *ShardErrors, error)` | Deletes ACLs matching the builder's filter. |

### Helper functions

`kadm` includes generic helpers for extracting single results from variadic responses:

```go
// Get the single topic response when you only created one topic.
resp, err := kadm.AnyE(admcl.CreateTopics(ctx, 3, 1, nil, "my-topic"))

// Get the first element from a slice response.
first, err := kadm.FirstE(admcl.ListGroups(ctx))
```

| Function | Description |
|---|---|
| `Any[M](m M) (V, bool)` | Returns the first value from a map response. |
| `AnyE[M](m M, err error) (V, error)` | Like `Any` but propagates the error. |
| `First[S](s S) (T, bool)` | Returns the first element from a slice response. |
| `FirstE[S](s S, err error) (T, error)` | Like `First` but propagates the error. |
